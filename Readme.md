## Now

### Create a new project from templates:

```sh
$ ~ npm install -g now
$ ~ now init nextjs
# or just run `now init` to see all templates
```
### And deploy it:

```sh
$ ~ cd nextjs
$ ~/nextjs now
```

### login on Terminal

```sh
$ now login
# mailアドレスを入力して送られてきたメールを確認すれば認証完了
```

### Deployment

```sh
koukibuu3 mh_bot $ now
```
